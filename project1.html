Academic Record Storage Using Blockchain

Project Duration: Jul 15, 2023 - Nov 30, 2023

This project aims to secure academic records using blockchain technology, providing a decentralized and tamper-proof system.

    Key Skills:

    • Blockchain Technology
    • Data Security
    • Analytical Thinking
    • Ethereum
    • Solidity
    • Truffle Suite
    • Javascript
    • Project Management

    Project Description

  • Developed a blockchain system for academic record storage, enhancing security and accuracy for educational institutions.
  • Implemented Ethereum blockchain and Solidity for smart contracts, Truffle for development, Ganache for testing, and MetaMask for transactions.
  • Contributed to cryptographic solutions and smart contract development, achieving a secure, user-friendly WebApp interface.
